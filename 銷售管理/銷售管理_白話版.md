# 銷售管理白話版：用日常例子理解銷售的藝術

## 前言：銷售就在你的日常生活中

銷售不是什麼神祕的東西。你每天都在看到銷售發生。當你進便利商店，店員說「我們新進的咖啡機不錯喔」，這是銷售。當房仲帶你看房子，詳細解釋社區的優勢，這是銷售。當你上Amazon看到商品推薦，那也是銷售。

這一章，我們不用複雜的商業詞彙，而是用你日常生活中最常見的例子，來理解銷售到底是怎麼運作的。

---

## 第1章：認識銷售 - 從夜市老闆開始

### 場景1：夜市賣烤香腸的老闆

想像你在夜市擺夜間的香腸攤位。

**第一步：認識你的客戶**
你觀察進來的人：有年輕上班族、有帶小孩的家庭、有學生。每群人的需求不同。
- 上班族可能累了，想吃點東西。
- 家庭可能找便宜又能飽的食物。
- 學生看到便宜才買。

**第二步：根據客戶調整你的銷售**
- 對上班族：「這個香腸新鮮現烤，加起司和洋蔥更香。」（賣品質）
- 對家庭：「3條打七折，很划算喔。」（賣價格）
- 對學生：「買10條以上可以再打折。」（賣量）

**第三步：增加信任**
- 讓客戶試吃一小口
- 告訴他：「我用的是進口香腸，很多人回頭客。」
- 跟常客打招呼，講一些輕鬆的話

**第四步：推動購買**
- 「現在買還送辣椒醬」
- 「晚來就沒有了」

**第五步：建立長期關係**
- 記得常客的喜好：「老兄，今天還是老樣子？」
- 偶爾給優惠：「今天人少，送你一條。」
- 讓他想下次還要來

**銷售公式就這麼簡單：**
了解客戶 → 提供解決方案 → 建立信任 → 成交 → 維護關係

### 場景2：房仲帶看房子

現在想像你是房仲，要賣一間房子。

**第一步：了解買家**
你問客戶：
- 預算多少？
- 想要幾房？
- 為什麼要買房？
- 什麼時候要住進去？

這個叫「需求分析」。不做這個，你就會浪費彼此的時間。

**第二步：推薦合適的房子**
根據客戶需求，你帶他看有潛力的房子。你說：
- 「這間在捷運站走5分鐘，你上班很方便。」
- 「社區管理好，保安24小時。」
- 「隔壁新開的商圈，房價升值潛力大。」

這個叫「提供解決方案」。

**第三步：消除疑慮**
客戶可能說：「位置不錯，但感覺有點舊？」
你回答：「沒錯，但正是因為舊，房價便宜。而且改裝的空間很大。房子位置好最重要，裝修以後再慢慢來。」

這個叫「處理異議」。

**第四步：催化成交**
- 「這間房子已經有其他人看過，有買方在考慮。」
- 「銀行現在貸款利率低，再晚就可能上升。」

這個叫「製造緊迫感」。

**第五步：成交後繼續服務**
- 幫忙聯絡律師、銀行
- 有問題就回電
- 未來賣房子，他會記得你

這就是為什麼房仲做的好的能存活很久。

## 第2章：B2B vs B2C - 批發和零售的差別

### B2C銷售：7-11 便利商店店員

**特徵1：決策很快**
你走進7-11，決定買什麼只要5分鐘。沒有複雜的審批流程。

**特徵2：客戶很多**
一間店一天可能服務1000個客戶。每個客戶的金額不大（通常100-500元），但量很大。

**特徵3：銷售的重點**
- 方便性：「買咖啡送集點卡」
- 價格：「促銷，一買一送」
- 衝動：「新產品試試看」

**店員的銷售技巧：**
```
顧客進來
  ↓
店員打招呼：「歡迎光臨」
  ↓
推薦新品或熱銷品：「我們新上市的便當滿好吃的」
  ↓
顧客買或不買
  ↓
成交或結束（通常只要2-3分鐘）
```

### B2B銷售：企業賣軟體給另一家企業

**特徵1：決策很慢**
可能要3-6個月。不是一個人決定，而是一個小組（IT主管、財務主管、使用部門主管）都要同意。

**特徵2：客戶很少**
一家軟體公司可能只有100個客戶，但每個客戶年度合約可能是100萬。

**特徵3：銷售的重點**
- ROI（投資報酬率）：「用我們的軟體，你能省30%的人力成本」
- 降低風險：「我們有金融業的成功案例，系統很穩定」
- 信任和關係：「我會派團隊進來幫你導入」

**B2B銷售過程：**
```
初始接觸（打電話或參展認識）
  ↓（2週）
初步會議：了解他們的需求
  ↓（2週）
提出提案：詳細的解決方案和價格
  ↓（1個月）
試用期：讓客戶用系統試試看
  ↓（2週）
談判價格和合約條款
  ↓（1週）
簽約成交
```

整個過程可能要3-6個月。

### 快速對比表

| 項目 | B2C (便利商店) | B2B (企業軟體) |
|------|---------------|---------------|
| 決策時間 | 2-5分鐘 | 3-6個月 |
| 客戶數 | 很多（日均1000人） | 很少（年100個） |
| 單筆金額 | 小（100-500元） | 大（幾十萬到幾百萬） |
| 購買者 | 個人 | 一個小組 |
| 銷售重點 | 方便、價格、體驗 | ROI、穩定、支持 |

## 第3章：銷售漏斗 - 像漏油一樣，客戶會掉出去

### 理解銷售漏斗

想像一個漏斗：
```
         潛在客戶 1000人
         /            \
        /              \
      轉換率20%
      /                \
    200人有興趣
    /                  \
   /                    \
轉換率30%
  /                      \
 60個機會
 /                        \
/                          \
轉換率50%
  /                          \
 30個成交
```

在每個階段，客戶都在掉出去。銷售管理就是減少掉出去的比例。

### 四個階段的例子：飲料店加盟

假設你要賣飲料店加盟。

**階段1：初始潛客（1000人）**
- 來源：Facebook廣告、加盟展、轉介
- 這些人只是知道有這個機會，沒有真正的動機

**階段2：有興趣的潛客（200人，轉換率20%）**
- 他們填了表格，或回應了廣告
- 你和他們有過初步接觸
- 但他們還沒有真正認真考慮

**活動：電話聯絡**
「您好，感謝您對我們加盟機會感興趣。我們現在在北部有幾間成功的門店，想邀您來了解一下。」

**階段3：符合條件的潛客變成「機會」（60個，轉換率30%）**
- 他們有預算（最少200萬投資）
- 他們有時間管理門店
- 他們真的在認真考慮

**活動：帶他們看示範店**
- 介紹成功的加盟主
- 秀營業額和利潤
- 解釋流程和支持

**階段4：成交（30個，轉換率50%）**
- 他們簽約了
- 你開始導入和培訓

### 如何改善轉換率

如果你的轉換率太低（比如每個階段只有10%），你需要找出為什麼：

1. **潛客品質不好** → 改變廣告目標，或提高填表的條件
2. **你的銷售話術有問題** → 訓練銷售員，改進說服方式
3. **你的產品有問題** → 改進加盟條件（更便宜、更好的支持等）
4. **競爭對手更強** → 強調你的優勢（比如更好的系統、更高的獲利）

## 第4章：CRM - 一本記錄所有客戶的筆記本

### 什麼是CRM？用日常例子

想像你是一個銀行經理，有1000個客戶。以前你用筆記本記筆記：

```
2024年1月：小王來辦房貸，聊到他兒子剛出生
2024年2月：小王要買投資型保險，上次聊過兒子，我可以賣兒童保險
2024年6月：小王要換房子，之前的房貸可以轉貸
```

這樣手動管理很困難。現在用CRM系統（客戶關係管理軟體），所有資訊都在電腦裡：

**CRM系統裡存什麼：**
- 客戶的聯絡資訊（電話、郵件、地址）
- 客戶購買歷史（買了什麼、買過幾次、金額多少）
- 每次接觸記錄（什麼時候打過電話、聊了什麼、他說了什麼）
- 後續行動（約定什麼時候跟進）

**好處：**
1. **不會忘記客戶** - 系統提醒你「小王上次聊過兒子，今天記得問一下」
2. **更快速成交** - 你知道客戶過去買過什麼，可以推薦相關產品
3. **團隊協作** - 小王這次和A銷售員聊，下次和B銷售員聊，B也能看到之前的紀錄，不用重新問一遍
4. **數據分析** - 看出什麼客戶最有價值，什麼銷售員最厲害

### CRM的功能展示

假設你用的是HubSpot（一個很受歡迎的CRM）：

**功能1：聯絡人管理**
- 存客戶的基本資訊（名字、電話、郵件、公司）
- 紀錄客戶的特殊情況（預算多少、決策者是誰、偏好什麼時間聊）

**功能2：活動紀錄**
- 他上次電話聊什麼
- 他上次看了你什麼產品
- 他上次說的疑慮是什麼

**功能3：自動提醒**
- 明天記得跟小王確認
- 小王3個月沒接觸了，記得主動聯絡

**功能4：銷售漏斗可視化**
你可以看一個圖表，顯示：
- 潛客池：50個人
- 初步接觸：15個
- 提案中：8個
- 談判中：3個
- 等著成交：1個

這樣你就知道什麼時候會有成交，計畫會不會達到。

### 選擇CRM的建議

不要一開始就選太複雜的系統。建議：

**剛開始（< 100個潛客）**
- 用Excel或Google Sheet就夠
- 簡單的欄位：客戶名字、電話、需求、預期成交日

**成長期（100-500個潛客）**
- 用Pipedrive或HubSpot的免費版
- 開始追蹤銷售漏斗

**大企業（> 1000個客戶）**
- 用Salesforce或HubSpot的付費版
- 支援複雜的工作流和報告

## 第5章：銷售績效獎金 - 怎麼激勵銷售員

### 銷售員為什麼需要獎金？

銷售是靠結果吃飯。如果只給固定薪水，銷售員就沒動力衝業績。所以要用獎金制度。

### 最常見的獎金設計

**方案1：基本薪資 + 傭金**

例如：
- 基本薪資：NT$40,000/月
- 傭金率：成交金額的5%

小王這月成交NT$1,000,000，他拿：
- 基本薪資：40,000
- 傭金：1,000,000 × 5% = 50,000
- 總收入：90,000

**方案2：基本薪資 + 獎金**

例如：
- 基本薪資：70,000
- 月配額目標：NT$500,000
- 達成100% → 獎金5,000
- 達成110% → 獎金10,000
- 達成120% → 獎金20,000

小王成交NT$600,000（達成120%）：
- 基本薪資：70,000
- 獎金：20,000
- 總收入：90,000

### 激勵制度設計的要點

**1. 目標要現實**
- 不要訂得太高，銷售員永遠達不到，會放棄
- 也不要太低，沒意義

經驗法則：60%的銷售員應該能達成目標

**2. 獎金要及時**
- 最好每月發獎金，這樣銷售員馬上有成就感
- 如果年底才發，太久了，動力會減弱

**3. 獎金結構要簡單**
如果計算方式太複雜，銷售員搞不懂，也沒用：
```
複雜：「達成100-110%時，傭金3%；110-120%時，傭金4%，但要扣除返品...」
簡單：「每成交NT$100,000，獎金NT$5,000」
```

**4. 要看重品質，不只數字**
有時候盲目追求數字會有問題。比如：
- 銷售員為了達成配額，賣給不符合的客戶，之後客戶要求退貨
- 或者用不誠實的手段成交

所以要加入品質指標：
- 3個月內沒退貨 → 加獎金
- 客戶滿意度 > 8分 → 加獎金

## 第6章：四種銷售風格

### 風格1：強硬推銷 (Hard Selling)

這是最傳統的銷售方式。

**怎麼做：**
銷售員不停說產品的優點，催促客戶買：
- 「這個功能多厲害！」
- 「現在買還有折扣，晚了就沒有。」
- 「其他人都買了，你為什麼不買？」

**適用場景：**
- 便宜的消費品（零食、飲料）
- 時間很短的銷售（車展、百貨週年慶）

**缺點：**
- 容易得罪客戶
- 客戶可能事後後悔，給差評

### 風格2：顧問式銷售 (Consultative Selling)

銷售員先幫客戶診斷問題，再推薦解決方案。

**怎麼做：**
```
1. 深入問問題：「貴公司現在的工作流程是怎樣？」
2. 找出痛點：「所以每月要花10天做這個流程？那肯定很浪費時間。」
3. 推薦方案：「我們的軟體可以自動化這個流程，能省50%的時間。」
4. 展示價值：「省下的時間，你團隊可以做更重要的事。」
```

**適用場景：**
- B2B銷售（企業軟體、諮詢服務）
- 複雜產品（機械、系統）
- 高單價銷售

**優點：**
- 客戶感覺你是在幫他，不是推銷
- 建立信任，容易成為長期客戶

### 風格3：解決方案銷售 (Solution Selling)

這是顧問式銷售的進階版。銷售員不只提供產品，而是提供完整的解決方案。

**例子：**
一個餐飲企業說：「生意不好，我不知道是因為菜色不好，還是服務不夠好，還是宣傳不足。」

強硬推銷員會說：「買我們的POS系統，管理更好。」

解決方案銷售員會說：
1. 先診斷：「我們先幫你做客流分析和顧客滿意度調查。」
2. 再提方案：「根據數據，你的問題有三個：(1)菜色不夠創新 (2)服務品質不穩定 (3)宣傳太少。」
3. 提供完整方案：「我們可以幫你：(1)帶進新的菜單(2)升級POS系統提高效率 (3)幫你做社群宣傳。」
4. 成長合作：「這不是一筆買賣，而是三個月的合作，我們要確保你生意好轉。」

### 風格4：SPIN提問法

這個方法強調用問題引導客戶發現自己的需求。

SPIN是四種問題：
- **Situation（情況）**：「貴公司現在用什麼系統？」
- **Problem（問題）**：「有沒有遇過數據錯亂的情況？」
- **Implication（影響）**：「這種錯亂會怎樣影響業績？」
- **Need-Payoff（需求滿足）**：「如果能解決這個問題，對你有多重要？」

**好處：** 客戶會自己說出需要你的理由，而不是你硬推給他。

## 第7章：銷售週期 - 從第一次接觸到簽約要多久？

### 什麼是銷售週期？

銷售週期就是從客戶第一次聽說你，到他真正簽約為止，要多久時間。

不同行業的銷售週期差很大：

| 行業 | 銷售週期 | 例子 |
|------|---------|------|
| 快消品 | 1-2週 | 買咖啡：幾分鐘 |
| 保險 | 2-4週 | 買意外險：1-2次會面 |
| 房地產 | 1-3個月 | 買房子：看房-貸款-簽約 |
| 企業軟體 | 3-6個月 | 買ERP系統：評估-試用-簽約 |
| 工業設備 | 6-12個月 | 買生產線：設計-製造-安裝 |

### 怎麼加快銷售週期？

**方法1：減少決策步驟**
- 以前要經過5個會面才能成交
- 改成3個會面，用視訊會議取代某些面談

**方法2：事前準備**
- 第一次會面就準備好完整的提案，不要到第二次才提
- 提前準備試用帳號

**方法3：清除障礙**
- 客戶為什麼猶豫？通常不是產品問題
  - 可能是「怕簽合約後被綁住」 → 提供30天試用期
  - 可能是「怕導入很麻煩」 → 承諾派人幫他導入
  - 可能是「怕貴」 → 提供分期付款

**方法4：製造緊迫感**
- 「這個月簽約有20%折扣，下月就沒有。」
- 「我們現在有人力幫你導入，再過兩週人力滿了。」

（但要小心，太假的緊迫感會讓客戶反感）

### 為什麼要關心銷售週期？

舉例：一個銷售員平均銷售週期是4個月，配額是NT$1,200,000。

這代表他每月平均成交NT$300,000。
- 1月：100萬成交（去年12月開始的客戶）
- 2月：30萬成交（1月初開始的客戶）
- 3月：30萬成交

他會發現，前面幾個月業績看起來不好，但其實他一直在努力，只是要等4個月才看到成果。

所以公司要耐心，不要在第2-3個月就說「你不行」。

同時，銷售員也要提早開始行動。如果銷售週期是4個月，現在的客戶是2個月前接觸的。所以他現在要已經在談下一批客戶，為3-4個月後準備。

## 第8章：客戶成功 - 成交不是結束，是開始

### 為什麼要關心客戶成功？

傳統想法：銷售員的工作是成交，成交後就交給客服。

新想法：成交後才是真正的關係開始。

**原因很簡單：**
- 獲得一個新客戶成本高（要花廣告費、銷售員時間）
- 保持現有客戶成本低（只要提供好服務）
- 現有滿意客戶會帶來新客戶（推薦、口碑）

**數字說明：**
如果你每年流失10%的客戶，要達到20%的成長，需要得到30%的新客戶。
但如果把流失率降到5%，只需要25%的新客戶就能達到20%成長。

所以留住客戶比拼命找新客戶重要。

### 客戶成功的四個階段

**階段1：購買後立即行動（第1-2週）**
- 感謝客戶：送禮物、親筆信、CEO打電話
- 快速導入：派人幫他開始用系統
- 確保滿意：「有什麼問題嗎？」

例子：Apple買了iPhone的人會收到歡迎郵件，裡面教怎麼開啟和基本設定。

**階段2：持續支持（第1-3個月）**
- 定期檢查：每週跟進一次，看他用得怎樣
- 提供訓練：教他更多功能
- 解決問題：他用不習慣的地方要幫他

例子：軟體公司會提供每週的線上訓練課程，教客戶新功能。

**階段3：成果驗證（第3-6個月）**
- 量化成果：「你現在的效率比導入前提升30%」
- 深化使用：「現在可以試試進階功能」
- 擴展銷售：「既然已經用了CRM，要不要試試我們的行銷自動化？」

例子：Salesforce會每季度提供業績報告，顯示客戶節省了多少時間。

**階段4：續約和擴展（第11-12個月）**
- 續約：確認他要繼續用下去
- 擴展：賣他更多產品或更多套數

例子：SaaS軟體會在合約快要期滿時聯絡客戶，問他續不續約，並推薦升級到更高級的方案。

### 客戶健康度評估

不是所有客戶都一樣。你要分辨哪些客戶健康，哪些客戶有風險。

**綠色客戶（健康）：**
- 定期在用系統
- 上次會面時他說很滿意
- 已經推薦其他人來用

**黃色客戶（有風險）：**
- 一個月沒登錄過系統
- 上次聊天時他提到預算吃緊
- 有意見但沒主動說

**紅色客戶（要掉了）：**
- 已經說要取消合約
- 發過幾次投訴郵件沒人回應
- 已經在找替代方案

你的工作是：
- 綠色客戶 → 定期聊天，推薦升級
- 黃色客戶 → 主動聯絡，解決問題，提醒價值
- 紅色客戶 → 緊急救援，可能要 CEO 親自出馬

## 第9章：通路管理 - 怎麼把產品賣給更多人

### 直銷 vs 代理商 vs 零售

**直銷（Direct Sales）**
- 你直接賣給客戶
- 優點：掌控完整體驗，利潤100%
- 缺點：成本高，很難覆蓋所有地區
- 例子：蘋果官網、Hermes精品店

**經銷商（Distributor）**
- 你賣給一個經銷商，他再賣給客戶
- 優點：覆蓋範圍廣，降低成本
- 缺點：利潤要分，客戶體驗你掌控不了
- 例子：某品牌的授權經銷商

**零售商（Retailer）**
- 你的產品在百貨、超市、實體店販賣
- 優點：一次上架可以賣到幾十家店
- 缺點：要付他們進場費，而且他們會要求很高的利潤
- 例子：康是美、屈臣氏

### 怎麼選擇通路？

**考慮因素：**

1. **你的產品特性**
   - 如果是高單價、需要詳細說明 → 直銷
   - 如果是便宜的快消品 → 零售通路

2. **你的目標市場**
   - 如果客戶集中在都市 → 零售
   - 如果客戶分散在各地 → 經銷商

3. **你的預算**
   - 如果預算多 → 直銷+零售
   - 如果預算少 → 經銷商

4. **競爭狀況**
   - 如果競爭激烈 → 要有多通路
   - 如果你產品獨特 → 一個通路就夠

### 經銷商管理

如果你用經銷商來銷售，要做什麼？

**1. 選對經銷商**
- 他們有自己的銷售團隊嗎？
- 他們的客戶基礎是什麼？
- 他們有沒有賣過類似產品？

**2. 訂定明確的協議**
- 他們負責的地區是哪裡？
- 他們每月要賣多少？
- 他們的利潤是多少？
- 你要提供什麼支持？

**3. 定期檢查和支持**
- 每月看他們的銷售數字
- 提供產品訓練
- 提供行銷物料幫他們宣傳
- 如果他們不達標，要想辦法幫他們

**4. 激勵和獎勵**
- 達成目標給獎金
- 特別好的表現給額外獎勵
- 高度成長的經銷商要重點培育

## 第10章：數據和銷售預測

### 銷售數據有多重要？

很重要，因為數據會告訴你：
- 這月會成交多少錢
- 哪個銷售員最厲害
- 哪類客戶最有價值
- 哪個產品最好賣

### 基本的銷售指標

**1. 銷售額**
最簡單的指標。「這月賣了NT$500,000」。

**2. 成交數**
賣了幾筆交易。有時候很多小單比一個大單更重要。

**3. 平均交易金額**
總金額 ÷ 交易數。如果你的平均從10萬升到15萬，代表你在賣更值錢的產品。

**4. 新客戶數**
這月獲得多少新客戶。重要，因為新客是未來的營收。

**5. 客戶保留率**
去年的客戶今年還有多少留下來。100個客戶留下80個 = 80%保留率。

### 怎麼預測下月的銷售？

方法1：看漏斗
- 進行中的機會：NT$500,000
- 根據過去的轉換率（50%），預期成交：NT$250,000

方法2：看歷史
- 去年1月賣100萬
- 今年成長20%
- 今年1月預期120萬

方法3：看銷售員的說法
- 問每個銷售員：「你這月預期成交多少？」
- 加起來就是預期數字

通常是三個方法都用，然後取一個平衡的數字。

### 為什麼預測要準？

如果預測不準，會發生什麼？

- 預測太樂觀 → 實際營收沒達到 → CEO失望，股東失望
- 預測太悲觀 → 實際營收超過 → 浪費了機會

所以銷售管理的一項重要工作是：確保預測準確。

怎麼做？
- 定期更新漏斗（每週）
- 聽銷售員的詳細說明，不只看數字
- 根據實際看到的進展調整預測
- 月末時回顧，看哪些預測準、哪些沒準，找出改善空間

## 第11章：常見的銷售問題和解決方案

### 問題1：銷售員說目標不現實

**銷售員說：**「NT$1,000,000 的月目標太高了，市場沒有那麼多機會。」

**怎麼判斷他說得對不對：**
1. 看去年的數據：他去年有沒有達過這個數字？
2. 看其他銷售員：其他人能不能達到？
3. 看市場：市場是不是真的在萎縮？

**如果他說得對，應該：**
- 調降目標到現實的水準
- 幫他分析為什麼達不到（是產品沒競爭力？是他銷售技巧不足？）

**如果他只是找藉口，應該：**
- 給他額外的訓練
- 更仔細地監督他的活動（電話數、會面數、提案數）
- 可能要替換他

### 問題2：銷售員一直沒有達成目標

**可能的原因：**
1. **他沒有足夠的潛客**
   → 幫他改善潛客來源（廣告、轉介、冷拜訪）

2. **他的轉換率太低**
   → 聽他和客戶的電話通話，找出問題
   → 給他銷售技巧訓練

3. **他給的價格太高**
   → 檢查競爭狀況
   → 考慮調降價格或改變策略

4. **他本人能力不足**
   → 可能需要替換他
   → 或轉調到較簡單的職位

### 問題3：銷售員常常超額達成目標

**這看起來是好事，但要小心：**
1. **他可能是靠不誠實的手段成交**
   - 承諾他做不到的東西
   - 隱瞞重要資訊
   → 這會導致客戶後來投訴或退貨

2. **他可能是賣給「不合格」的客戶**
   - 客戶的預算不足
   - 客戶實際不需要這個產品
   → 之後客戶會要求退貨

**怎麼判斷？**
- 看他的客戶保留率（是不是常常有人退貨）
- 看他的客戶滿意度（NPS、評論）
- 定期聽他的銷售電話

**解決方案：**
- 讚美他，但提醒他品質也重要
- 給他更高級的客戶，考驗他的真實能力
- 可能他真的就是個銷售高手，應該升職

## 第12章：銷售與行銷要合作

### 銷售和行銷為什麼常常吵架？

**銷售員的想法：**
「行銷給我的潛客都很爛。這些人根本沒有買的意思，我花時間跟進都浪費。」

**行銷人的想法：**
「銷售員都只會抱怨。我送給他們的潛客品質很好，是他們銷售技巧不夠，跟進不足。」

兩邊都覺得另一邊沒有幫忙。結果是：
- 行銷亂生成潛客
- 銷售員不想跟進
- 公司的轉換率很低
- 營收沒有成長

### 怎麼讓銷售和行銷合作？

**第1步：定義「好的潛客」**
銷售和行銷坐在一起，討論什麼樣的潛客值得跟進：
- 公司規模多大？
- 預算大約多少？
- 有什麼樣的需求？

這個叫「MQL」（Marketing Qualified Lead，符合條件的行銷潛客）。

**例子：**
「我們只要規模50人以上、年營收1000萬以上、需要CRM系統的公司。如果不符合這些條件，不值得行銷浪費錢去吸引。」

**第2步：行銷只生成符合定義的潛客**
- 改變廣告目標
- 提高潛客表格的要求
- 不要為了多潛客數量就降低品質

**第3步：銷售承諾跟進每一個符合條件的潛客**
- 保證在72小時內聯絡
- 定期反饋（「跟進了10個潛客，3個有機會，7個沒有」）

**第4步：定期同步會議**
- 每週：「這週新來的潛客是什麼品質？」
- 每月：「這月轉換率怎樣？」
- 每季度：「我們的策略要不要調整？」

**第5步：共同的獎勵**
- 不要只獎勵銷售的成交數
- 也要獎勵行銷的轉換率
- 他們達成目標，一起分享獎金

這樣銷售就不會怨行銷，行銷也會在乎銷售的成交率。

## 第13章：常見的銷售騙招和為什麼不要用

### 騙招1：製造虛假的緊迫感

**例子：**
「這個優惠只到今天，明天就沒有了。」
但其實他每週都這樣說。

**為什麼不要用：**
- 客戶很聰明，會發現你在騙
- 他會失去信任，不再相信你的任何承諾
- 他會跟朋友說「這家的銷售員喜歡唬爛」

### 騙招2：隱瞞產品的缺點

**例子：**
「這個軟體可以自動化所有流程。」
但其實有些很複雜的流程還是要手工。

**為什麼不要用：**
- 成交後客戶會發現，覺得上當
- 會要求退貨或退款
- 會給你負評

### 騙招3：承諾做不到的事

**例子：**
「簽約後我們會幫你完全客製化系統。」
但其實公司根本沒有那個人力和能力。

**為什麼不要用：**
- 導入時做不到，客戶大失所望
- 會投訴到你的主管
- 可能導致合約被取消

### 正確的銷售態度

**要老實：**
- 「這個產品有這些優點，但也有這些限制。」
- 「我們可以做到這些，這些我們做不到。」
- 「如果我不確定，我回公司問一下再告訴你。」

**為什麼老實反而更有效：**
1. 客戶會信任你
2. 成交的客戶會滿意（因為期望管理得好）
3. 滿意的客戶會推薦你
4. 長期看，誠實會帶來更多生意

有句話說得好：「短期的騙可能成交，長期的誠實才能留住客戶。」

## 結語：銷售的本質

讀完這章，你應該發現：銷售的本質其實很簡單。

它不是複雜的技巧或心理戰。它就是：
1. **理解客戶的真實需求**
2. **提供真誠的解決方案**
3. **建立信任**
4. **成交**
5. **確保客戶滿意**
6. **保持長期關係**

無論你是在夜市賣香腸，還是在辦公室賣軟體，這個流程都一樣。

最成功的銷售員，通常不是最會說話的人，而是最誠實、最在乎客戶的人。

祝你銷售成功！
