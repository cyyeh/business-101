## 🎓 關鍵概念速查

### 核心概念
- **供應鏈管理**：統籌原材料採購→生產→配送→最終用戶的全過程
- **長鞭效應**：下游需求小幅波動 → 上游訂單波動放大
- **及時制（JIT）**：減少庫存，訂單來時才補貨
- **延遲策略**：推遲製造/配送時機，直到客戶需求確認

### 重要模型
- **Kraljic採購矩陣**：高/低價值 × 高/低風險 → 4象限採購策略
- **EOQ經濟訂購量**：平衡訂購成本與持有成本的最優批量
- **ABC庫存分類**：20%商品貢獻80%價值，實施差異管理
- **S&OP銷售與營運規劃**：整合市場需求與生產能力

### 現代趨勢
- **數位化**：IoT、區塊鏈、AI預測、數位孿生
- **韌性**：多元化供應、戰略庫存、應急預案
- **永續**：碳中和、循環經濟、ESG責任
- **地緣政治**：脫鉤、友岸外包、近岸化
