# 銷售管理練習與自測

## 使用說明

本章包含多種題型，用來測試你對銷售管理的理解。建議按順序完成，並在查看答案前先自己思考。

## 第一部分：是非題（10題）

請判斷以下敘述是「正確」還是「錯誤」。

### 題目1
銷售員的主要目標是成交盡可能多的交易，以達成月度配額。

**A) 正確**
**B) 錯誤**

<details>
<summary>點擊查看答案</summary>

**答案：B) 錯誤**

**解釋：**
雖然成交交易是銷售員的重要工作，但更重要的是成交「高品質」的交易。如果銷售員為了達成配額而：
- 向不符合條件的客戶推銷
- 承諾做不到的事情
- 隱瞞產品缺點

這些交易會在事後導致客戶投訴、退貨或取消合約，最終損害公司。

正確的做法是：重視交易品質、客戶滿意度和長期客戶價值。

</details>

### 題目2
B2B銷售通常比B2C銷售的銷售週期短。

**A) 正確**
**B) 錯誤**

<details>
<summary>點擊查看答案</summary>

**答案：B) 錯誤**

**解釋：**
- **B2B銷售**：決策周期長（3-6個月），因為涉及多層審批和複雜的評估
- **B2C銷售**：決策周期短（數分鐘到數週），因為是個人消費決策

例如：
- 買手機（B2C）：可能只要30分鐘決定
- 買企業軟體（B2B）：可能要3個月才能通過審批

所以B2B銷售週期通常是B2C的10倍以上。

</details>

### 題目3
CRM系統的主要價值是追蹤銷售員的工作時間。

**A) 正確**
**B) 錯誤**

<details>
<summary>點擊查看答案</summary>

**答案：B) 錯誤**

**解釋：**
CRM系統的主要價值不是監督銷售員，而是：
- 集中客戶資料，方便查詢
- 記錄客戶互動歷史
- 自動化重複工作
- 支持數據分析和預測
- 改善客戶體驗

如果把CRM當成監控工具，銷售員會抗拒，導致數據不準確。好的CRM應該是幫助銷售員工作更有效率的工具。

</details>

### 題目4
銷售漏斗的轉換率越高，代表銷售策略越成功。

**A) 正確**
**B) 錯誤**

<details>
<summary>點擊查看答案</summary>

**答案：A) 正確**（但需要限定條件）

**解釋：**
銷售漏斗轉換率高確實是成功的象徵，但必須確保：
1. **潛客品質好** - 高轉換率不代表東西賣得好，可能只是因為一開始篩選得很嚴格
2. **客戶生命週期價值高** - 應該看長期而不是只看成交數
3. **客戶滿意度高** - 成交後客戶沒有退貨或投訴

所以正確的說法是：「在保證客戶品質的前提下，轉換率越高越好。」

</details>

### 題目5
銷售激勵制度應該只以成交金額為指標。

**A) 正確**
**B) 錯誤**

<details>
<summary>點擊查看答案</summary>

**答案：B) 錯誤**

**解釋：**
只以成交金額為指標會導致：
- 銷售員推銷給不符合的客戶
- 承諾虛假的承諾
- 忽視客戶滿意度

好的激勵制度應該包括：
- 成交金額（40%）
- 成交率和銷售週期（20%）
- 客戶滿意度（20%）
- 重複購買率和保留率（20%）

這樣才能鼓勵銷售員追求品質和長期關係。

</details>

### 題目6
Salesforce和HubSpot是銷售管理軟體中的唯一選擇。

**A) 正確**
**B) 錯誤**

<details>
<summary>點擊查看答案</summary>

**答案：B) 錯誤**

**解釋：**
市面上有很多優秀的CRM和銷售管理軟體，包括：
- Pipedrive（簡單好用，適合中小企業）
- Microsoft Dynamics（與Excel整合，適合Microsoft使用者）
- Zoho（功能完整，價格實惠）
- Freshsales（新興但功能強大）

選擇應該根據：
- 公司規模
- 預算
- 現有系統
- 業務需求

不是大公司才好，而是「適合你」才好。

</details>

### 題目7
顧問式銷售總是比強硬推銷更有效。

**A) 正確**
**B) 錯誤**

<details>
<summary>點擊查看答案</summary>

**答案：B) 錯誤**

**解釋：**
不同的銷售場景需要不同的銷售風格：

- **強硬推銷適用於：**
  - 便宜的消費品（飲料、零食）
  - 時間很短的決策（超市結帳）
  - 衝動購買的商品

- **顧問式銷售適用於：**
  - 複雜的產品（企業軟體）
  - 高單價商品
  - 需要長期諮詢的服務

用錯銷售風格反而會失效。比如在便利商店用顧問式銷售，客戶會覺得浪費時間。

</details>

### 題目8
客戶成功管理（Customer Success）是客服部門的責任，不是銷售的責任。

**A) 正確**
**B) 錯誤**

<details>
<summary>點擊查看答案</summary>

**答案：B) 錯誤**

**解釋：**
客戶成功管理應該是整個公司的責任：

- **銷售的責任：**
  - 設定正確的期望（不要oversell）
  - 成交後交接給成功團隊
  - 定期檢查客戶進展

- **客服的責任：**
  - 技術支持和問題解決
  - 日常培訓和指導

- **成功團隊的責任：**
  - 主動檢查客戶健康度
  - 確保客戶達成業務目標
  - 尋找擴展機會

銷售不能說「成交後就不管了」。長期客戶價值需要整個團隊的協力。

</details>

### 題目9
銷售與行銷的衝突是正常的，不需要刻意協作。

**A) 正確**
**B) 錯誤**

<details>
<summary>點擊查看答案</summary>

**答案：B) 錯誤**

**解釋：**
銷售和行銷的協作（Smarketing）是現代企業必須的：

如果他們不協作：
- 行銷會浪費預算生成低品質潛客
- 銷售會抱怨沒有好潛客
- 整個轉換漏斗效率低

協作的好處：
- 定義明確的「合格潛客」(MQL)標準
- 建立共同的目標和KPI
- 定期溝通和同步
- 共享數據和洞察

HubSpot的成功案例正是因為他們把銷售和行銷整合在一個平台上。

</details>

### 題目10
銷售預測的目的是評估銷售員的績效，而不是預測公司的營收。

**A) 正確**
**B) 錯誤**

<details>
<summary>點擊查看答案</summary>

**答案：B) 錯誤**

**解釋：**
銷售預測的主要目的是：
- **預測公司營收** - CFO需要知道這月會賺多少，好規劃成本
- **規劃資源** - 人力、庫存、現金流規劃
- **評估進度** - 與目標相比，目前進度如何

評估銷售員績效用的是別的指標，比如：
- 配額達成率
- 成交率
- 平均銷售週期

銷售預測應該是準確的，不應該被用來評估個人績效（那樣銷售員會為了保護自己而低報）。

</details>

### 題目11
一個銷售員的銷售週期平均是4個月，配額是NT$1,200,000。她現在是1月中旬，才完成NT$300,000（目標NT$1,000,000）。她說「別擔心，3月會有大單。」這個說法對嗎？

**A) 完全對，4個月週期意味著1月的營收來自去年10月的工作**
**B) 完全錯，她應該這月達成目標**
**C) 對一半，她說得對但應該再提前開始行動**
**D) 無法判斷，需要看她目前有多少進行中的機會**

<details>
<summary>點擊查看答案</summary>

**答案：A) 完全對，4個月週期意味著1月的營收來自去年10月的工作**

**解釋：**
這是一個很重要的概念。銷售週期不只影響成交，還影響績效評估的方式。

如果銷售週期是4個月：
- 1月的成交來自 → 9月開始的客戶
- 2月的成交來自 → 10月開始的客戶
- 3月的成交來自 → 11月開始的客戶

所以1月只達成目標的30%是很正常的。她現在看起來業績不好，但其實她9月時應該已經在努力了，只是要等4個月才看到成果。

重點是：她現在應該已經在開發會在4月-6月成交的客戶了。

這也解釋為什麼長銷售週期的行業（房地產、企業軟體）需要跨年度評估績效，而不是只看月度。

</details>

### 題目12
你是銷售主管，發現銷售員A的成交率很高（月配額NT$500,000，常常達成NT$700,000），但客戶投訴率也很高。你應該怎麼做？

**A) 一定要開除他，因為高投訴代表有欺騙**
**B) 讚美他，然後深入了解問題所在**
**C) 降低他的配額，因為他的客戶品質差**
**D) 增加他的配額，因為他很厲害**

<details>
<summary>點擊查看答案</summary>

**答案：B) 讚美他，然後深入了解問題所在**

**解釋：**
高成交 + 高投訴可能有幾種原因：

1. **他真的用不誠實的手段成交** （最糟情況）
   - 需要調查並改正

2. **他的客戶要求很高** （可能是好事）
   - 也許他賣給的是高要求的客戶
   - 只要他解決問題，客戶最後會滿意

3. **他的導入或產品有問題** （不是他的錯）
   - 也許是你的產品或服務有問題
   - 不是他的銷售問題

怎麼做：
- 聽他和客戶的銷售通話
- 看他的提案内容
- 跟投訴的客戶聊天（了解真實原因）
- 根據調查結果決定

可能的結果：
- 如果他用的是誠實的方式，就讚美他，教他如何處理高要求的客戶
- 如果他虛假承諾，就要改正或替換他

別急著下判斷，而是調查。

</details>

### 題目13
你的B2B軟體公司銷售給企業，平均客戶取得成本(CAC)是NT$50,000，客戶終身價值(LTV)是NT$150,000。這個比率怎樣？

**A) 很差，LTV應該至少是CAC的5倍**
**B) 不錯，LTV是CAC的3倍，表示生意可持續**
**C) 很好，比率越低越好**
**D) 無法判斷，需要看行業平均值**

<details>
<summary>點擊查看答案</summary>

**答案：B) 不錯，LTV是CAC的3倍，表示生意可持續**

**解釋：**
LTV:CAC比率是衡量銷售效率的重要指標。

- **比率 < 1:1** = 虧本（獲得一個客戶的成本 > 他帶來的利潤）
- **比率 1:1 - 2:1** = 有盈利但不健康（邊際很小）
- **比率 3:1** = 很好，這個業務可持續
- **比率 5:1 或更高** = 非常好，但要確保CAC降不下去了

在這個例子中：
- CAC = NT$50,000（花NT$50,000才獲得一個客戶）
- LTV = NT$150,000（客戶在整個週期帶來NT$150,000利潤）
- 比率 = 3:1

這表示：每花1元獲得客戶，能賺3元的利潤。在B2B軟體行業，3:1被認為是健康的。

所以這個業務是健康的，可以繼續投資。

</details>

### 題目14
哪種銷售策略最適合當客戶已經知道問題，但還不知道最好的解決方案？

**A) 強硬推銷 - 快速推銷你的產品**
**B) 顧問式銷售 - 幫助他評估各種選項**
**C) Challenger Sale - 挑戰他對問題的認知**
**D) SPIN銷售 - 用提問引導他發現解決方案**

<details>
<summary>點擊查看答案</summary>

**答案：B) 顧問式銷售 - 幫助他評估各種選項**

**解釋：**
不同的銷售策略適用於不同的情況：

| 情況 | 最佳策略 | 理由 |
|------|---------|------|
| 客戶不知道有問題 | Challenger Sale | 需要教育和改變認知 |
| 客戶知道問題，不知道解決方案 | 顧問式銷售 | 需要評估和建議 |
| 客戶知道問題，已有初步認知 | SPIN銷售 | 引導他自己發現需求 |
| 低接觸銷售（便宜商品） | 強硬推銷 | 快速決策，減少思考 |

在題目的場景中：
- 客戶已知「我需要提高效率」
- 但不知道「用CRM還是ERP還是別的」

此時顧問式銷售最有效，因為你需要：
1. 了解他的具體情況
2. 幫他評估各種選項的優缺點
3. 推薦最適合的解決方案

</details>

### 題目15
一個初創新產品，應該用「直銷」還是「經銷商」？

**A) 一定是直銷，經銷商會搶你的利潤**
**B) 一定是經銷商，擴張更快**
**C) 看情況，取決於產品複雜度和市場覆蓋需求**
**D) 兩種同時用，分散風險**

<details>
<summary>點擊查看答案</summary>

**答案：C) 看情況，取決於產品複雜度和市場覆蓋需求**

**解釋：**
選擇銷售通路要考慮多個因素：

| 因素 | 傾向直銷 | 傾向經銷商 |
|------|---------|----------|
| 產品複雜度 | 高（需詳細說明） | 低（簡單明白） |
| 銷售週期 | 長（需要深入關係） | 短（交易導向） |
| 單價 | 高 | 低 |
| 市場覆蓋 | 集中 | 分散 |
| 公司規模 | 大（有資金養銷售團隊） | 小（資金有限） |
| 品牌控制 | 重要 | 次要 |

舉例：
- **Tesla（高單價、複雜產品、品牌重要）** → 直銷是對的
- **飲料公司（低單價、簡單、需覆蓋全國）** → 經銷商是對的

對初創公司：
- 如果資金有限，用經銷商擴張
- 如果是高階產品需要品牌控制，用直銷但在特定市場先試
- 理想情況是「逐步過渡」：先直銷建立品牌，再加經銷商擴張

</details>

### 題目16
你是HR經理，想設計一個銷售激勵制度。以下哪個是最平衡的？

**A) 100%基本薪資，沒有獎金（公平但沒動力）**
**B) 100%傭金，沒有基本薪（動力強但不穩定）**
**C) 70%基本薪資 + 30%獎金（平衡穩定和動力）**
**D) 看公司文化，有的偏穩定，有的偏動力**

<details>
<summary>點擊查看答案</summary>

**答案：C) 70%基本薪資 + 30%獎金（平衡穩定和動力）** （或 D 也有理）

**解釋：**
不同的銷售環境需要不同的激勵結構：

| 結構 | 基本薪資% | 獎金% | 適用情景 |
|------|----------|-------|---------|
| 100% 傭金 | 0% | 100% | 房仲、保險（個人貢獻） |
| 50/50 | 50% | 50% | 傳統銷售 |
| 70/30 | 70% | 30% | 帳戶管理、企業銷售 |
| 80/20 | 80% | 20% | 客戶成功管理 |
| 100% 基本 | 100% | 0% | 內勤銷售、客服混合 |

最平衡的考量：
- **70/30**給了銷售員穩定感（70%保證），也有動力（30%獎金）
- 不會因為市場不好就沒收入（比100%傭金好）
- 也不會因為沒獎金就沒衝勁（比100%基本好）

但答案D也對，因為「最平衡」取決於：
- 公司文化
- 銷售員類型（老手 vs 新手）
- 行業慣例
- 產品特性（易賣 vs 難賣）

一個好主管會針對不同銷售員調整（老手可以100%傭金，新手70/30）。

</details>

### 題目17
一個客戶簽了年度合約，已經用了3個月。現在她說很滿意，但最近有點忙，用得比較少。你是客戶成功經理，應該怎麼做？

**A) 放著不管，因為他已經簽約不會退出**
**B) 主動聯絡，了解他為什麼用得少，是否需要幫助**
**C) 等到11月再聯絡，準備續約對話**
**D) 主動推薦升級到更高級的方案**

<details>
<summary>點擊查看答案</summary>

**答案：B) 主動聯絡，了解他為什麼用得少，是否需要幫助**

**解釋：**
這是一個「黃色客戶」信號：
- 使用頻率下降 ⚠️
- 但還沒有明確表示不滿 ✓

不處理會變成「紅色客戶」（高風險）的原因：
- 他習慣用得少，可能逐漸忘記系統的價值
- 3個月後會評估「這個工具值不值得年續」
- 等到11月才聯絡，太晚了

正確的做法：
1. **立即主動聯絡**「我發現你最近用得比較少，想了解一下有沒有什麼困難」
2. **了解真實原因**
   - 是用不習慣？→ 提供額外訓練
   - 是功能不符需求？→ 調整使用方式或升級
   - 就是忙？→ 提醒他還有我們在這

3. **提早開始續約對話** - 9月就開始聊，而不是11月

這樣可以：
- 提高客戶滿意度和使用度
- 降低9月就可能損失的流失
- 建立長期關係

答案D不對是因為不應該在還沒確保他滿意的情況下就推銷升級。

</details>

### 題目18
一個B2B銷售部門，平均銷售週期4個月，月配額NT$1,000,000，團隊有5個銷售員。現在是6月底，管理漏斗中的機會金額如下：
- 進行中(10%概率)：NT$2,000,000
- 談判中(30%概率)：NT$3,000,000
- 簽約前(70%概率)：NT$2,000,000
預期7月營收應該是多少？

**A) NT$1,000,000（月配額）**
**B) NT$1,700,000（30%+70%的加權）**
**C) NT$1,600,000（各階段機會×概率加權）**
**D) 無法判斷，需要更多資訊**

<details>
<summary>點擊查看答案</summary>

**答案：C) NT$1,600,000（各階段機會×概率加權）** 但實際應該是 **NT$2,200,000**

**解釋：**
漏斗基礎預測的計算：

```
進行中（10%概率）：NT$2,000,000 × 10% = NT$200,000
談判中（30%概率）：NT$3,000,000 × 30% = NT$900,000
簽約前（70%概率）：NT$2,000,000 × 70% = NT$1,400,000
預期7月營收 = NT$200,000 + NT$900,000 + NT$1,400,000 = NT$2,500,000
```

但題目答案給的選項沒有完全正確。如果一定要選，應該是：

**如果考慮現實調整：**
- 「進行中」的概率往往被高估（通常要再打個折扣）
- 「簽約前」的概率相對準確

調整後的預測：
```
進行中：NT$2,000,000 × 5% = NT$100,000
談判中：NT$3,000,000 × 25% = NT$750,000
簽約前：NT$2,000,000 × 70% = NT$1,400,000
= NT$2,250,000 (約 NT$2,200,000)
```

這樣預期7月會超配額（NT$2,200,000 > NT$1,000,000 × 1人 = 應該是5人 = NT$5,000,000？）

等等，題目可能有邏輯問題。如果5人配額是NT$5,000,000/月，而漏斗只有NT$7,000,000，預期營收只有NT$2.2-2.5M，就遠低於配額。這表示：
- 後續要加強潛客開發
- 或者現在漏斗的下一階段機會不足

**重點學習：**
1. 要定期檢查漏斗的「總機會金額」
2. 預期營收應該 ≥ 月度配額，否則會miss目標
3. 銷售主管要提早識別潛客不足的問題，提前行動

</details>

### 題目19
Salesforce在CRM市場成功的最主要原因是什麼？

**A) 功能最完整**
**B) 價格最便宜**
**C) 簡單易用，能快速導入，降低成本**
**D) 有最多的合作夥伴**

<details>
<summary>點擊查看答案</summary>

**答案：C) 簡單易用，能快速導入，降低成本**（但D也有理）

**解釋：**
Salesforce的成功不是因為功能最多或最便宜，而是因為他改變了銷售管理軟體的交付方式：

**傳統CRM的問題：**
- 要3-6個月才能導入
- IT人員要花很多時間配置
- 銷售員討厭用，因為太複雜
- 總成本很高

- SaaS模式，不需要本地安裝
- 一週內就能開始使用
- 介面相對簡單，銷售員容易上手
- 按月訂閱，不需要大筆投資

這意味著：
- 小企業也能用
- 導入成本低
- 銷售員願意用（好用 = 好數據 = 好決策）

這對銷售管理的啟示：
好的工具應該是「易用」而不是「功能最多」。銷售員不會因為功能多就用，但會因為簡單好用而愛用。

### 題目20
你正在進行一個大型客戶的銷售，已經談了2個月，客戶說「我們對你的方案很感興趣，但還要和財務部討論預算。」這時你應該怎麼做？

在B2B銷售中，「採購委員會」(buying committee) 往往包括多個人：
- 使用部門經理（說要什麼功能）
- IT主管（檢查技術相容性）
- 財務主管（檢查預算）
- 高階主管（最終決策）

不能只跟一個人聊。要找出所有決策者，和他們都建立關係。

### 情景1：初創銷售部門

你是一家初創技術公司的銷售主管，公司成立3個月，有2個銷售員。現在遇到問題：
- 銷售員A：每月有5個初始接觸，但沒有一個轉化成機會
- 銷售員B：初始接觸少（月2個），但轉化率高（50%轉成機會）

你應該如何輔導他們？

<details>
<summary>點擊查看答案</summary>

**分析與建議：**

**銷售員A的問題：** 初始接觸多但轉化率為0%，代表：
1. **潛客品質差** - 可能在和不符合條件的人說話
2. **銷售技巧差** - 有機會但沒能引起興趣
3. **產品或話術有問題** - 可能在說錯誤的價值主張

**對A的輔導：**
- 檢查他的潛客來源：他是怎麼找到這5個人的？品質怎樣？
- 聽他的銷售通話：他在說什麼？是否清楚地傳達了價值？
- 教他SPIN銷售或顧問式銷售：不要直接推銷，先問問題了解需求
- 如果還是不行，可能需要考慮換人

**銷售員B的問題：** 初始接觸少，代表：
1. **潛客開發能力差** - 不知道怎麼找潛客
2. **主動性不足** - 只等公司提供潛客
3. **可能太專注於品質** - 寧願少接觸也要高品質

**對B的輔導：**
- 讚美他的轉化率，但鼓勵他提高初始接觸數
- 教他潛客開發技巧：冷電話、領英、轉介等
- 如果他是新手，他的品質導向是對的
- 但隨著經驗增加，應該擴大初始接觸數

**整體策略：**
```
銷售員A：量多但質差 → 改善品質
銷售員B：量少但質好 → 增加量

理想狀態：兩人都是「量多質好」
```

**長期建議：**
- 定期追蹤每個銷售員的漏斗在各階段的轉化率
- 根據轉化率低的階段提供特定訓練
- 考慮設置佣金制度鼓勵他們擴大初始接觸

### 情景2：客戶流失危機

你的SaaS公司平時流失率很低（3-5%），但最近發現：
- 過去3個月，有3個大客戶要求取消合約（佔總營收的25%）
- 他們的共同點：都說「功能不符合需求」，但在簽約前都說「很滿意」

這發生了什麼？你應該怎麼做？

<details>
<summary>點擊查看答案</summary>

**診斷：**

這個情況很常見，通常反映了「期望管理不佳」：

1. **銷售時的過度承諾**
   - 銷售員為了成交，說了產品實際做不到的功能
   - 或誇大了產品的能力

2. **導入不足**
   - 簽約後沒有充分幫助客戶上手
   - 客戶發現產品不如預期，而沒有人幫他

3. **客戶成功管理缺失**
   - 沒有人主動檢查客戶是否達成業務目標
   - 等到客戶失望了，已經太晚了

**立即行動：**

1. **調查和拯救**（本週）
   - 親自打電話給這3個客戶，了解具體是什麼功能不符
   - 問：「我們有辦法解決嗎？或者能不能調整合約條款？」
   - 目的：能救就救，不能救也要了解原因

2. **內部檢查**（本週）
   - 聽銷售錄音，看銷售員是否承諾了不該承諾的
   - 檢查簽約後的導入流程，是否充分
   - 檢查客戶成功團隊的活動，為什麼沒有及早識別問題

3. **流程改進**（本月）
   - 建立「簽約後30天檢查」流程
   - 客戶成功經理要在導入後1週、1月、3月檢查客戶狀態
   - 設置「黃色客戶預警」機制（使用頻率下降、沒達成KPI等）

4. **銷售員培訓**（本月）
   - 重申誠實銷售的重要性
   - 教銷售員「如何承諾他們能保證的」
   - 把客戶流失也算入銷售員的績效評估（不要只看成交數）

5. **客服機制**（本月）
   - 提高客服響應速度
   - 確保技術問題能及時解決

**根本解決：**
銷售和客戶成功必須協作。不能是「銷售成交了就交棒」。而是：
- 銷售負責「承諾正確的期望」
- 客戶成功負責「實現期望」
- 如果實現不了，要及早識別並改正

</details>

### 情景3：競爭對手下價

你的B2B銷售團隊平時成交率還不錯，但最近發現：
- 一個新競爭對手進入市場，價格只有你的60%
- 你的銷售員開始失利，客戶說「你們太貴了」

該怎麼辦？

<details>
<summary>點擊查看答案</summary>

**分析：**

「客戶說太貴」往往不是真實原因，而是理由。真實原因可能是：
1. 客戶不相信你的價值
2. 銷售員沒有清楚傳達ROI
3. 產品確實不適合客戶
4. 對手產品真的更好

**不應該做的事：**
- 直接降價（會開啟價格戰，最後誰都賺不到錢）
- 放棄客戶（有些客戶值得競爭）

**應該做的事：**

1. **理解競爭對手**
   - 他們是怎麼這麼便宜的？功能是否少？
   - 客戶滿意度怎樣？
   - 他們的商業模式能持續嗎？

2. **重新審視自己的價值主張**
   - 你的產品為什麼比對手好？列出具體的差異點
   - 量化你的優勢的ROI
   - 如果真的沒有差異，就該改進產品

3. **分段定價**
   - 對預算有限的客戶，提供「精簡版」（功能少，價格低）
   - 對高階客戶，強調高級功能和支持

4. **改進銷售話術**
   - 不要防守「太貴」，而要進攻「值不值」
   - 「對手價格低，但他們有24小時客服嗎？有成功經理嗎？」
   - 比較「總成本」而不是「軟體成本」

5. **強調不可複製的優勢**
   - 如果是人力（客戶成功團隊），說「我們的人比他們更懂行業」
   - 如果是技術（AI預測），說「我們的預測準確度更高」
   - 如果是支持（響應時間），說「我們平均1小時回應」

6. **選擇性競爭**
   - 不是所有客戶都值得競爭
   - 如果客戶只看價格，他們也不是好客戶（容易流失）
   - 專注在「看重價值」的客戶

**具體銷售話術範例：**
```
客戶：「對手便宜40%」

錯誤回應：「我們也可以降價」
正確回應：「我理解價格是考量。不過你的年度軟體成本是NT$100萬，我們的平均ROI是30%（可省NT$300萬成本）。對手雖然便宜，但他們的ROI是多少？」
```

**長期策略：**
如果對手真的能持續低價，代表要麼他們做得更高效，要麼他們要虧損。
- 如果他們做得更高效，你應該學習
- 如果他們在虧損，他們遲早會倒
- 所以不必急著跟風降價

結論：競爭不是降價，而是找到自己獨特的價值，並讓客戶看見這個價值。

</details>

## 第五部分：延伸思考（3-4題）

這些題目沒有唯一答案，需要你的深入思考。

### 思考題1：銷售員和銷售主管的角色邊界

一個銷售員持續超額達成目標（月配額NT$1,000,000，常常達到NT$1,500,000），但他很不願意幫助其他銷售員、分享他的成功經驗。你應該怎麼辦？

**思考方向：**
- 他為什麼不願意分享？是自私、還是害怕競爭減少、還是沒有動力？
- 如何設計激勵讓他願意分享？
- 是否應該強制他分享，還是尊重他的個人選擇？
- 如何平衡「個人業績」和「團隊協作」？

<details>
<summary>參考思考方向</summary>

**可能的原因分析：**

1. **自私或競爭心理**
   - 他認為幫助別人會減少自己的相對優勢
   - 或他擔心別人學會後會威脅他的位置

2. **激勵制度的問題**
   - 激勵制度只獎勵個人成績，沒有獎勵分享或團隊成績
   - 所以他沒有動力幫助別人

3. **溝通問題**
   - 他可能不知道你期望他分享
   - 或者他覺得自己的方法難以教人

**解決方案：**

1. **調整激勵制度** ⭐
   - 個人成績佔60%，團隊成績/知識分享佔40%
   - 如果全隊達成目標，給額外獎金
   - 最佳銷售員（兼具成績和分享）才能升職

2. **建立分享文化**
   - 每月銷售會議時，要求成績好的銷售員分享技巧
   - 給分享者額外獎勵（現金、免費參加研討會等）
   - 不是強制，而是鼓勵

3. **管理對話**
   - 和他聊天：「你做得很棒。你願不願意幫助其他人也這麼棒？」
   - 解釋為什麼：「如果全隊都很強，公司會更成功，你的升職機會也更大」
   - 聽他的想法：「你害怕什麼嗎？」

4. **明確期望**
   - 告訴他「分享是這個角色期望的一部分」
   - 「頂級銷售員除了成績好，還要指導團隊」
   - 設定明確的目標（比如「每月帶一個銷售員打一通電話」）

5. **創造win-win**
   - 讓他成為官方導師，給他額外的頭銜和薪水
   - 這樣他既能保留高收入，也能得到認可

**關鍵洞察：**
一個超級銷售員不幫助團隊，他的邊際價值其實很有限。但如果他能帶動全隊，他的價值就翻倍了。作為主管的工作就是幫助他看見這一點，並調整激勵制度讓他有動力這麼做。

</details>

### 思考題2：如何評估一個銷售員是「真正厲害」還是「靠運氣」？

一個銷售員已經工作6個月，成交了3個大單（合計NT$500萬），配額月NT$300萬。但他還沒有在同一客戶上有重複購買或擴展。你怎麼判斷他的真實能力？

**思考方向：**
- 要追蹤哪些指標？
- 3個大單是真正的銷售能力，還是運氣？
- 如何測試他在不同情況下的表現？
- 長期看，什麼指標最能反映他的真實價值？

<details>
<summary>參考思考方向</summary>

**區分「厲害」和「運氣」的指標：**

| 指標 | 一次性運氣 | 真正厲害 |
|------|----------|--------|
| **成交數** | 3個大單就完成了 | 月月穩定有成交 |
| **平均周期** | 很短（可能巧合） | 穩定的平均周期 |
| **客戶品質** | 短期內沒有退貨就算贏 | 3-6個月後還在用、開心 |
| **重複購買** | 沒有 | 有客戶升級或擴展 |
| **客戶滿意度** | 不知道 | NPS > 50 |
| **管道深度** | 漏斗很空 | 漏斗裡有源源不斷的機會 |
| **銷售週期** | 波動很大 | 相對穩定 |
| **方法論** | 說不出為什麼成交 | 能複述銷售過程、方法一致 |

**6個月的表現評估：**

**資料點1：3個大單，NT$500萬**
- 不足以判斷他是否厲害
- 可能是運氣，也可能是真的
- 樣本太少

**關鍵問題（應該問他）：**
1. 這3個單子的銷售過程是什麼？他是怎麼找到他們的？
2. 他目前管道裡有多少機會？
3. 他有沒有和過去的客戶保持聯絡？
4. 他有沒有幫客戶升級或擴展的計畫？

**測試真實能力的方式：**

1. **持續觀察（未來6個月）**
   - 看他接下來能不能穩定每月NT$300萬
   - 如果3個月只有NT$100萬，代表可能是運氣

2. **聽銷售通話**
   - 他怎麼和客戶溝通？
   - 他用的是什麼銷售技巧？
   - 是否有一致的方法論，還是每次都靠臨時發揮？

3. **檢查客戶回饋**
   - 3個客戶現在滿意嗎？
   - 有沒有投訴？
   - 會不會推薦其他人來？

4. **測試他在挫折中的表現**
   - 給他一個比較難的客戶
   - 看他怎麼處理被拒絕
   - 真正厲害的銷售員能從失敗中學習

5. **看他的學習意願**
   - 他有沒有參加銷售訓練？
   - 有沒有看銷售書籍或Podcast？
   - 是否願意接受回饋？

**可能的結論：**

- **如果持續成交 + 客戶滿意 + 方法一致** → 真的厲害
- **如果3個月後只有1個月達標** → 可能是運氣
- **如果他說不出銷售過程 + 不想學習** → 可能只是靠際遇
- **如果客戶開始投訴 + 沒有重複購買** → 銷售素質有問題

**長期建議：**
不要因為一時的好成績就認為他很厲害。要看：
1. **穩定性** - 能否月月達成
2. **客戶品質** - 客戶滿不滿意
3. **可複製性** - 能否教別人做同樣的事
4. **成長性** - 能否持續學習進步

一個真正厲害的銷售員應該同時具備這4點。

</details>

### 思考題3：SaaS公司應該用「直銷」還是「合作夥伴/代理」？

你的B2B SaaS軟體現在只有直銷通路，客戶增長速度慢（每月新增5-10個客戶）。一個大的代理商說他們可以幫你一年內增加到50個客戶。但代理商會要求40%的利潤。你應該同意嗎？

**思考方向：**
- 直銷 vs 代理的優缺點？
- 40%利潤率真的太高嗎？
- 如何評估代理商的實際能力？
- 長期看，應該怎麼策略？

<details>
<summary>參考思考方向</summary>

**直銷 vs 代理商的權衡：**

| 維度 | 直銷 | 代理商 |
|------|------|-------|
| 擴張速度 | 慢（受人力限制） | 快（他們有團隊） |
| 成本 | 高（要養銷售隊伍） | 低（只在有成交時付費） |
| 品牌掌控 | 高 | 低（客戶體驗你掌控不了） |
| 利潤率 | 高（100% = 直接收客戶的費用） | 低（只能分60-70%） |
| 客戶服務品質 | 高（你的人） | 中等（代理可能不那麼認真） |
| 長期客戶關係 | 強（直接互動） | 弱（隔著代理） |

**定量分析：**

假設年訂閱費NT$100,000，毛利率40%：

**直銷場景：**
```
每月5個新客戶 × 12月 = 60個客戶/年
每客戶年毛利：NT$40,000
年毛利：60 × NT$40,000 = NT$2,400,000

但直銷成本（2個銷售員、1個經理）≈ NT$2,500,000
淨利 = NT$2,400,000 - NT$2,500,000 = -NT$100,000（虧損）
```

**代理商場景：**
```
代理幫你一年帶來50個客戶
他要40%利潤，你拿60%
每客戶年毛利：NT$40,000
你的淨利：50 × NT$40,000 × 60% = NT$1,200,000

直銷成本省下來，只需維持關鍵人力：
省下 NT$2,500,000，現在只需 NT$500,000（管理 + 客戶成功）
淨利 = NT$1,200,000 - NT$500,000 = NT$700,000（盈利）
```

**結論：從財務角度，代理商更划算。**

但還有其他考量：

**風險1：代理商的真實能力**
- 他說「一年50個客戶」，但能否兌現？
- 怎麼驗證他的承諾？

**應該問他：**
- 過去做過類似產品嗎？成績怎樣？
- 他們的銷售團隊有多少人？
- 你的產品在他們的優先級是多少？
- 他們願不願意簽「最少客戶數保證」？

**風險2：品牌和客戶體驗**
- 代理商可能不了解你的產品
- 客戶成交後可能體驗不好
- 客戶會直接怪你，不會怪代理商

**應該做的：**
- 給代理商充分的訓練
- 規定銷售話術和品牌承諾
- 自己負責客戶導入和成功（不讓代理商碰）

**風險3：客戶所有權**
- 如果代理商把客戶「綁」了，未來會很難
- 可能變成「帮代理商打工」

**應該談清楚：**
- 合約條款：客戶跟你，不跟代理商
- 如果代理商跑了，客戶誰負責

**長期策略建議：**

最好的方案是「分階段」：

**第1階段（現在）：試點代理商**
```
和1-2個代理商簽試點合約（6個月）
看他們是否能真正帶來客戶
同時，自己繼續做直銷，維持增長
```

**第2階段（6個月後）：評估成果**
```
如果代理商表現好 → 擴大合作範圍和數量
如果代理商表現差 → 縮小範圍或終止合作
```

**第3階段（1年後）：多通路並行**
```
直銷 + 多個代理商 + 其他通路（轉介、自營等）
分散風險，最大化成長
```

**最終建議：**
1. **不要一次性把所有希望放在代理商身上**
2. **簽短期試點合約，而不是長期綁定**
3. **自己保留直銷能力，不要完全依賴代理**
4. **明確定義成功指標（客戶數、留存率、滿意度）**
5. **把客戶關係掌控在自己手上，不讓代理商獨佔**

這樣，既能快速擴張，也能保持對品牌和客戶的掌控。

</details>
