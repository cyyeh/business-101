# 營運管理：練習與自測

> 本練習涵蓋本主題的核心概念，包括是非題、選擇題、名詞配對、情境分析與計算應用。建議先自行作答，再參考答案解析進行檢討。

## 一、概念快速檢測（是非題）

### 題目

1. **生產力（Productivity）是指單位時間內的產出量，與產能（Capacity）沒有區別。**

2. **經濟訂購量（EOQ）模型假設需求是恆定且可預測的，這個模型在需求不穩定時無法使用。**

3. **精實生產（Lean Manufacturing）的目標是完全消除庫存，達到零庫存狀態。**

4. **六標準差（Six Sigma）方法中，DMAIC循環的「控制」階段是最後一步，完成後不需要再回到「定義」階段。**

5. **Just-In-Time（JIT）系統要求供應商頻繁小量交貨，這會增加供應商的成本，但對製造企業有利。**

6. **服務營運與製造營運的根本區別是服務無法庫存，因此必須100%準確預測需求。**

7. **制約理論（Theory of Constraints）認為系統的整體績效受限於最弱的環節（瓶頸）。**

8. **統計過程控制（SPC）是一種事後檢驗方法，用來檢查已完成的產品是否符合標準。**

9. **銷售與營運規劃（S&OP）主要是銷售部門的職責，生產部門只需執行計畫。**

10. **供應鏈設計中，採用多個供應商會增加企業的談判成本，因此單一供應商是最優選擇。**

### 答案與解析

| 題號 | 答案 | 解析 |
|------|------|------|
| 1 | **X 錯誤** | 產能（Capacity）是理論上的最大產出量；生產力（Productivity）是實際達到的產出量或產出與投入的比率。兩者不同。 |
| 2 | **O 正確** | EOQ假設需求穩定，當需求波動大時，需要用更複雜的動態規劃或啟發式方法。 |
| 3 | **X 錯誤** | 精實生產目標是消除浪費，而非完全零庫存。安全庫存和週期庫存仍然是必要的，但應最小化。 |
| 4 | **X 錯誤** | 六標準差DMAIC是循環流程，「控制」階段之後還應持續監控，當發現新問題時會再回到「定義」階段，開啟新一輪改善。 |
| 5 | **O 正確** | JIT要求頻繁小量交貨，增加了供應商的運輸與協調成本。但透過長期合作與信息共享，整個供應鏈的整體成本反而降低。 |
| 6 | **X 錯誤** | 雖然服務無法庫存，但透過容量預訂、排隊管理等手段，可以部分平衡供需，不需要100%精確預測。 |
| 7 | **O 正確** | 這是TOC的核心原理。系統輸出受限於最弱環節的產能，改善其他部分不會提升整體績效。 |
| 8 | **X 錯誤** | SPC是過程控制方法，應用在製程中監控，及時發現異常，屬於預防性質而非事後檢驗。 |
| 9 | **X 錯誤** | S&OP是跨職能的高層管理流程，需要銷售、生產、財務等部門協同決策，不是單方面職責。 |
| 10 | **X 錯誤** | 多供應商有風險分散、競爭壓力、創新來源的優勢，應根據物料重要性選擇。單一供應商在某些情況下有效率優勢，但缺乏冗餘性。 |

| 1 | **D** | 利用率 = 實際產能 / 有效產能 = 800/850 ≈ 94%；效率 = 有效產能 / 理論產能 = 850/1000 = 85%。|

| 9 | **C** | 不做任何調整讓缺貨發生是不負責任的。正確做法應該是A、B、D之一或組合。 |

## 四、情境分析題

### 情境一：快速成長公司的產能困境

**背景：**
某科技公司生產軟體授權，過去三年年成長率50%。今年銷售預測為200萬套，但目前客服團隊只能處理100萬套的安裝、培訓和支援。客服團隊說需要1年時間才能招聘和培訓新員工到熟手水準。

**問題：**
1. 這是什麼性質的瓶頸？（短期vs長期，結構性vs非結構性）
2. 根據制約理論的五聚焦步驟，應該如何處理？
3. 可能的短期方案有哪些？

**參考答案：**

1. **瓶頸性質：**
   - 結構性瓶頸（不是暫時的，是系統設計的問題）
   - 長期瓶頸（需要1年才能解除）
   - 人力資源瓶頸（員工培訓時間長）

2. **五聚焦步驟應用：**
   - 第一步：確認客服是瓶頸 ✓（已識別）
   - 第二步：最大化現有客服的效率
     - 優化服務流程，減少單個客戶所需時間
     - 建立自助服務平台
     - 優先服務高價值客戶
   - 第三步：使其他部門服從瓶頸
     - 銷售部門調整銷售節奏，不要短期內簽太多大合約
     - 開發部門配合客服的負載
   - 第四步：突破瓶頸
     - 加快員工招聘（可能需要提高薪資）
     - 外包部分客服工作
     - 外聘顧問加快培訓
   - 第五步：回到第一步
     - 解除客服瓶頸後，可能開發或銷售變成新瓶頸

3. **短期方案（3-6個月）：**
   - 流程優化：標準化常見問題解答，減少處理時間20-30%
   - 自動化：建立聊天機器人處理常見問題
   - 外包：外包非核心支援工作
   - 靈活用工：聘請兼職或臨時員工加班處理高峰期

---

### 情境二：供應商品質問題

**背景：**
某汽車零件製造廠，80%的零件由一家供應商提供。最近該供應商因產能滿載，開始外包一部分訂單給第二家廠商。結果交貨的零件不良率從1%飆升至8%，導致組裝線停工，客戶開始投訴。

**問題：**
1. 這個情況違反了供應鏈管理的哪些原則？
2. 應該採用什麼樣的供應商策略來防止這類問題？
3. 短期和長期的解決方案是什麼？

**參考答案：**

1. **違反的原則：**
   - 單一供應商的風險：沒有冗餘性，供應商的問題直接影響整個製造廠
   - 供應商能力評估不足：沒有預留產能安全係數
   - 信息溝通不足：供應商應該提前通知要外包，製造廠應該評估資格

2. **改進的供應商策略：**
   - 多供應商策略：維持2-3家合格供應商，分散80%的訂單為：
     - 主供應商：60%
     - 次供應商：30%
     - 備用供應商：10%
   - 供應商認證：對所有供應商（包括二級供應商）進行品質認證
   - 長期合作協議：明確說明不允許未授權外包

3. **解決方案：**
   - **短期（1-2個月）：**
     - 與現有供應商密集溝通，立即停止第三方外包
     - 增加檢驗抽樣，確保進廠零件品質
     - 調整組裝線進度，消化缺陷零件

   - **長期（3-12個月）：**
     - 開發第二家供應商，完成認證和小批量測試
     - 逐步轉移30%訂單到新供應商
     - 與主供應商協商增加產能投資（購置新設備、聘用員工）
     - 實施供應商品質改善計畫（六標準差或Kaizen）

## 五、計算與應用題

### 問題一：安全庫存計算

**題目：**
某醫療器材公司銷售一種醫療設備。根據過去12個月的歷史資料：
- 月均需求：1000台
- 需求標準差：150台
- 供應商前置時間：2個月
- 目標服務水準：95%（Z值為1.645）

請計算：
1. 安全庫存（Safety Stock）
2. 再訂購點（Reorder Point）
3. 若要將服務水準提升至99%（Z值為2.326），安全庫存將增加多少？

**解答：**

**安全庫存公式：**
```
SS = Z × σ × √LT

其中：
Z = 服務水準對應的標準正態分數
σ = 需求的標準差（月）
LT = 前置時間（月）
```

1. **95%服務水準的安全庫存：**
```
SS = 1.645 × 150 × √2
   = 1.645 × 150 × 1.414
   = 349台（約）
```

2. **再訂購點：**
```
ROP = 月均需求 × 前置時間 + 安全庫存
    = 1000 × 2 + 349
    = 2,349台

解釋：當庫存降至2,349台時，應該下單，這樣在供應商2個月內送達時，
庫存恰好降至349台的安全庫存
```

3. **99%服務水準的安全庫存：**
```
SS99% = 2.326 × 150 × √2
      = 2.326 × 150 × 1.414
      = 494台

增加量 = 494 - 349 = 145台（約41%增加）
```

**管理意涵：**
- 服務水準提升4%（從95%到99%），需要增加145台庫存
- 企業需在品質保證和庫存成本間權衡
- 對於生命攸關的醫療設備，99%甚至99.9%的服務水準可能是合理的

### 問題二：制約理論應用

**題目：**
某家具製造廠有三個主要工作中心：
- 切割中心（Cutting）：產能60件/天
- 組裝中心（Assembly）：產能50件/天
- 油漆中心（Painting）：產能55件/天

目前訂單充足，每天應接訂單100件（每件都要經過三個工序）。

問題：
1. 哪個工作中心是瓶頸？
2. 目前的實際產出是多少？
3. 根據TOC，應該如何處理非瓶頸工作中心？
4. 解除瓶頸的投資是什麼？

**解答：**

1. **識別瓶頸：**
   - 切割：產能60件/天 > 需求100件/天？不夠
   - 組裝：產能50件/天 < 需求100件/天 ← **瓶頸**
   - 油漆：產能55件/天 > 需求100件/天？接近瓶頸

**組裝中心是瓶頸**，因為產能最低。

2. **實際產出：**
```
實際產出 = 瓶頸的產能 = 50件/天

即使訂單有100件，也只能生產50件。
利用率：
- 切割中心：50/60 = 83%（有閒置產能）
- 組裝中心：50/50 = 100%（滿載）
- 油漆中心：50/55 = 91%（有閒置產能）
```

3. **非瓶頸的處理：**
   - 切割中心只應生產50件/天（供應組裝需求），不要為了提高自身利用率而超額生產
   - 油漆中心只應生產50件/天（滿足組裝的輸出）
   - 三個中心應同步運作，避免組裝前產生庫存堆積

4. **突破瓶頸的方案：**

若要日產100件，需要提升組裝中心的產能。投資選項：

| 方案 | 成本 | 時間 | 效果 |
|------|------|------|------|
| 增加班次（加班） | $2000/天 | 立即 | 可增至60-70件/天 |
| 購置新組裝機器 | $200,000 | 3個月 | 可增至80-100件/天 |
| 簡化組裝流程（改善） | $50,000 | 2個月 | 可增至60-70件/天 |
| 外包部分組裝 | 變動成本$500/件 | 2-4週 | 可增至80-100件/天 |

**最優選擇：** 先用加班和流程改善達到70-75件/天（成本$52,000，時間2個月），再評估是否購置新設備達成100件/天。
