## 1. 個人財務與理財規劃 Personal Finance & Financial Planning

📖 [前往主題學習](#個人財務與理財規劃)

| 術語 | English | 定義 |
|------|---------|------|
| 複利 | Compound Interest | 利息滾入本金後再計算利息，產生「利滾利」效果 |
| 72 法則 | Rule of 72 | 以 72 除以年利率，估算資產翻倍所需年數 |
| 資產配置 | Asset Allocation | 根據風險承受度將資金分配到不同資產類別的策略 |
| 現代投資組合理論 | Modern Portfolio Theory, MPT | Markowitz 提出的透過分散投資降低風險的理論 |
| 效率前緣 | Efficient Frontier | 在特定風險水準下能達到最高預期報酬的投資組合集合 |
| 4% 法則 | 4% Rule | 退休後每年提領投資組合的 4%，可維持 30 年不耗盡的經驗法則 |
| 緊急預備金 | Emergency Fund | 用於應對突發狀況的流動性儲備，通常建議 3-6 個月生活費 |
| 定期定額投資 | Dollar-Cost Averaging, DCA | 固定金額定期投資，分散進場時機風險的策略 |
| 指數型基金 | Index Fund | 追蹤特定市場指數的被動投資基金 |
| 信用評分 | Credit Score | 評估個人信用風險的數值，影響借貸利率與額度 |
| 所得稅 | Income Tax | 對個人或企業所得課徵的稅款 |
| 資本利得稅 | Capital Gains Tax | 對出售資產所獲利潤課徵的稅款 |
| FIRE 運動 | Financial Independence Retire Early | 透過高儲蓄率與投資達到財務自由、提前退休的理念 |
| 淨值 | Net Worth | 個人所有資產減去負債後的總價值 |

## 2. 微觀經濟學基礎 Microeconomics Fundamentals

📖 [前往主題學習](#微觀經濟學基礎)

| 術語 | English | 定義 |
|------|---------|------|
| 供給法則 | Law of Supply | 價格越高，供給量越多的經濟規律 |
| 需求法則 | Law of Demand | 價格越低，購買量越多的經濟規律 |
| 市場均衡 | Market Equilibrium | 供給量等於需求量的價格與數量組合 |
| 價格彈性 | Price Elasticity | 商品需求量對價格變化的反應程度 |
| 邊際效用 | Marginal Utility | 額外消費一單位商品的額外滿足感 |
| 邊際成本 | Marginal Cost | 增加一單位產出所需的額外成本 |
| 完全競爭市場 | Perfect Competition | 眾多買賣家、商品同質、進出自由的市場結構 |
| 壟斷性競爭 | Monopolistic Competition | 眾多競爭者但商品存在差異化的市場結構 |
| 寡佔 | Oligopoly | 少數幾家大企業主導的市場結構 |
| 獨佔 | Monopoly | 單一賣家壟斷整個市場的結構 |
| 外部性 | Externality | 一方經濟活動對第三方產生的非市場影響 |
| 公共財 | Public Good | 具有非排他性與非競爭性的財貨 |
| 資訊不對稱 | Information Asymmetry | 交易一方掌握比另一方更多資訊的現象 |
| 賽局理論 | Game Theory | 分析相互影響環境中策略決策的數學框架 |
| 納什均衡 | Nash Equilibrium | 沒有參與者願意單方面改變策略的穩定狀態 |
| 消費者剩餘 | Consumer Surplus | 消費者願意支付的最高價與實際支付價格之差 |

## 3. 總體經濟學基礎 Macroeconomics Fundamentals

📖 [前往主題學習](#總體經濟學基礎)

| 術語 | English | 定義 |
|------|---------|------|
| GDP | Gross Domestic Product | 國家生產的最終商品與服務的市場價值總額 |
| 消費者物價指數 | CPI, Consumer Price Index | 追蹤消費品與服務價格變化的指標 |
| 核心通膨 | Core Inflation | 排除食品與能源後的通膨率 |
| 量化寬鬆 | Quantitative Easing, QE | 央行購買長期資產以增加貨幣供應量的政策 |
| 乘數效應 | Multiplier Effect | 政府支出的初始增加帶動更大產出增加的效應 |
| 景氣循環 | Business Cycle | 經濟的擴張、高峰、衰退、谷底循環過程 |
| 領先指標 | Leading Indicators | 預示經濟走向的先行指標 |
| 匯率 | Exchange Rate | 兩種貨幣的相對價值 |
| 購買力平價 | PPP, Purchasing Power Parity | 長期匯率應使兩國貨幣購買力相等的理論 |
| 摩擦性失業 | Frictional Unemployment | 勞工轉換工作過程中的暫時性失業 |
| 結構性失業 | Structural Unemployment | 產業轉型或技能不匹配造成的失業 |
| 菲利普曲線 | Phillips Curve | 描述失業率與通膨率負相關關係的模型 |

## 4. 全球金融體系 Global Financial System

📖 [前往主題學習](#全球金融體系)

| 術語 | English | 定義 |
|------|---------|------|
| 金融市場 | Financial Markets | 金融資產與證券交易的場所，包括股票、債券、外匯與衍生品市場 |
| 央行貨幣政策 | Central Bank Monetary Policy | 中央銀行通過調整利率與貨幣供應量以控制通膨與經濟增長 |
| 基礎貨幣 | Monetary Base | 央行發行的現金與商業銀行準備金，是金融系統的基礎 |
| 國際清算銀行 | Bank for International Settlements, BIS | 各國央行的銀行，制定國際銀行業監管標準 |
| 資本充足率 | Capital Adequacy Ratio | 銀行資本與風險加權資產的比例，衡量銀行風險承擔能力 |
| 金融穩定委員會 | Financial Stability Board, FSB | 協調各國金融監管政策以維護全球金融穩定的國際組織 |
| 流動性風險 | Liquidity Risk | 金融機構無法及時籌集資金以應對提現需求的風險 |
| 信用評級機構 | Credit Rating Agencies | 評估政府與企業信用風險的專業機構（標普、穆迪、惠譽） |
| 系統性風險 | Systemic Risk | 一家機構或市場的崩潰導致整個金融系統危機的風險 |
| 金融衍生品 | Financial Derivatives | 價值來自基礎資產（股票、債券、商品）的金融工具 |
| 套利 | Arbitrage | 利用不同市場的價格差異進行無風險交易獲利 |
| 借貸利差 | Lending Spread | 銀行借款利率與存款利率之間的差異，代表銀行利潤來源 |

## 9. 消費者行為與心理學 Consumer Behavior & Psychology

📖 [前往主題學習](#消費者行為與心理學)

| 術語 | English | 定義 |
|------|---------|------|
| 消費者決策過程 | Consumer Decision Process | 問題認知→資訊搜尋→方案評估→購買決策→購後行為的五階段模型 |
| 錨定效應 | Anchoring Effect | 決策時過度依賴最先接收到的資訊（錨點）的認知偏誤 |

| 社會認同 | Social Proof | 人們在不確定時傾向跟隨他人行為的心理傾向 |
| 沉沒成本謬誤 | Sunk Cost Fallacy | 因已投入無法回收的成本而繼續追加投資的非理性行為 |

| 推力理論 | Nudge Theory | Thaler 與 Sunstein 提出的透過設計選擇架構引導行為的理論 |
| 預設選項效應 | Default Effect | 人們傾向接受預設選項而非主動更改的現象 |
| 衝動購買 | Impulse Buying | 未經事前計畫、受情緒驅動的即時購買行為 |
| 神經行銷學 | Neuromarketing | 運用腦科學方法研究消費者對行銷刺激反應的領域 |
| 顧客旅程 | Customer Journey | 消費者從認知品牌到購買再到忠誠的完整體驗歷程 |

| 復原時間目標 | RTO, Recovery Time Objective | 系統必須恢復運作的最長時間要求 |
