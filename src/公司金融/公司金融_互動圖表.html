併購流程互動圖表

完整的併購交易分為四個階段，每個階段包含關鍵決策點和風險評估。

第一階段：戰略與初期評估

1.1 戰略規劃

• 確定併購目的（成長、協同、多元化）
• 識別潛在目標企業
• 初步價值評估
✗ 風險：戰略不清晰導致併購失敗

1.2 接洽與談判

• 聯繫目標企業管理層
• 簽署保密協議（NDA）
• 初步估值討論
✗ 風險：敵意收購觸發防禦

1.3 交割框架

• 主要交易條款討論
• 價格範圍協商
• 付款方式（現金/股票）
✓ 里程碑：MOU（意向書）簽署

第二階段：盡職調查與估值

2.1 財務盡職調查

• 審計歷史財務報表
• 驗證應收/應付帳款
• 評估或有負債
耗時：2-3 個月
✗ 風險：發現隱藏負債

2.2 法律盡職調查

• 審查主要合同
• 識別訴訟風險
• 驗證智慧財產權
耗時：2-3 個月
✗ 風險：發現合規問題

2.3 商業與估值

• DCF 估值分析
• 協同效應量化
• 敏感性分析
耗時：1-2 個月
✓ 里程碑：最終價格範圍確定

第三階段：融資與審批

3.1 融資結構

• 確定融資來源比例
• 現金 vs 股票 vs 債務
• 評估融資可行性
✗ 風險：融資無法完成

3.2 監管審批

• 反壟斷審查
• 產業監管許可
• 外資管制許可
耗時：3-12 個月
✗ 風險：被監管機構否決

3.3 股東大會

• 雙方股東投票
• 最後一次定價確認
• 簽署最終協議
✓ 里程碑：交割條件滿足

第四階段：交割與整合

4.1 交割與支付

• 最終交割成交
• 現金/股票支付
• 所有權轉移
✓ 里程碑：交易完成

4.2 PMI 第一階段

• 0-3 個月：關鍵決策期
• 整合團隊建立
• 快速行動消除冗餘
重要：人才保留期
✗ 風險：人才流失

4.3 PMI 持續改進

• 3-24 個月：系統整合
• 實現預期協同效應
• 文化融合
✓ 里程碑：協同目標達成

時間與成本總結：
• 小型併購（<$5B）：9-15 個月，成本 0.5-2% 交易價值
• 中型併購（$5-50B）：15-24 個月
• 大型併購（>$50B）：24-36 個月，涉及複雜監管審查
常見失敗點：盡職調查不足、監管反對、整合無力

M&A 成功的關鍵因素

階段 成功因素
戰略與評估 清晰的併購動機與合理估值
盡職調查 深入全面、及時發現隱藏風險
融資與審批 融資確定、監管順利、條款清晰
PMI 整合 快速決策、人才保留、協同兌現
