# 供應鏈管理：練習與自測

## 是非題（10題）

**題目 1**
供應鏈管理只涉及物流與倉儲。

**答案**：否

**解析**：供應鏈管理涵蓋原材料採購、生產、物流配送、銷售預測、庫存管理等多個環節，物流只是其中一部分。

---

**題目 2**
「及時制」（JIT）在所有情況下都能降低企業成本。

**答案**：否

**解析**：JIT在穩定、可預測的環境中有效，但在供應不確定、突發情況頻繁的環境中會增加風險。2020年疫情期間許多企業因過度採用JIT而面臨庫存不足。

---

**題目 3**
長鞭效應的主要原因是消費者購買行為的波動。

**答案**：否

**解析**：長鞭效應的根本原因是資訊延遲、訂單批量化決策、價格波動誘導囤積等供應鏈設計問題，而非純粹的消費者需求波動。

---

**題目 4**
供應商評估時，價格是唯一需要考慮的因素。

**答案**：否

**解析**：現代供應商評估應綜合考慮品質、交付、創新、財務健康、環保責任等多維度，單純以價格衡量會遺漏關鍵風險。

---

**題目 5**
ABC庫存分類法中，C類商品應該保持最少的安全庫存。

**答案**：否

**解析**：ABC分類中，C類商品（低價值、低頻銷售）應保持相對較高的安全庫存，因為缺貨的相對成本高；而A類商品應保持較低安全庫存以降低持有成本。

---

**題目 6**
區塊鏈技術可以完全解決供應鏈的透明性問題。

**答案**：否

**解析**：區塊鏈只能確保已記錄資訊的真實性與不可篡改性，但無法確保最初資訊的準確性（垃圾進垃圾出原則）。供應鏈透明性還需要依賴企業間的信任與資訊共享意願。

---

**題目 7**
推式（Push）供應鏈總是比拉式（Pull）供應鏈的成本更低。

**答案**：否

**解析**：推式供應鏈製造成本低但庫存成本高；拉式供應鏈庫存成本低但製造成本高。哪種更便宜取決於產品特性與市場環境。

---

**題目 8**
VMI（供應商管理庫存）意味著供應商完全負責客戶的庫存決策。

**答案**：否

**解析**：VMI是一種協作模式，供應商根據約定的庫存目標與客戶共享的銷售數據進行補貨決策，但客戶仍保持一定的監督與控制權。

---

**題目 9**
供應鏈的數位化只涉及採購與庫存管理系統。

**答案**：否

**解析**：供應鏈數位化涵蓋IoT傳感、區塊鏈追溯、AI預測、數位孿生、自動化等多個領域，影響從原材料到最後一哩配送的全過程。

---

**題目 10**
蘇伊士運河擱淺事件證明全球供應鏈應該轉向本地化生產。

**答案**：否

**解析**：該事件提醒我們應該進行地理多元化、建立備用運輸方案、增加戰略庫存，而非完全放棄全球化。完全本地化會喪失全球化的效率優勢。

---

**題目 1**
以下哪個不是供應鏈管理的核心目標？

A. 降低成本
B. 提高客戶滿意度
C. 增加市場佔有率
D. 提升供應鏈韌性

**答案**：C

**解析**：增加市場佔有率是企業戰略目標，不是供應鏈管理的直接目標。供應鏈管理的核心目標是通過優化流程支撐企業目標。

B. 戰略夥伴戰略
C. 槓桿採購戰略

**答案**：C

**題目 3**
經濟訂購量（EOQ）公式中，訂購頻次增加時，年度總成本會如何變化？

A. 總是增加
B. 總是減少
C. 先增後減，存在最優點
D. 保持不變

**答案**：C

**解析**：訂購頻次增加會降低持有成本（更新周期短）但增加訂購成本（次數多）。兩者的平衡點就是EOQ。

**題目 4**
若產品需求標準差σ=50件/月，訂購提前期L=10天，目標服務水準97.5%（Z=1.96），安全庫存應該是多少？

A. 10件
B. 20件
C. 35件
D. 50件

**答案**：C

**解析**：
安全庫存 = Z × σ × √L
σ(日) = 50/30 = 1.67件/日
安全庫存 = 1.96 × 1.67 × √(10/30) = 1.96 × 1.67 × 0.577 ≈ 1.9 × 18.5 ≈ 35件

## 情境分析題（3-4題）

### 情境一：鮮花供應鏈危機

背景：一家花卉批發企業與全國100家花店合作，供應鮮花。

**情景**：疫情導致進口鮮花被禁，企業需要改為採購本地種植花卉。但本地花農分散在20個不同地區，供應不穩定，某些品種（進口玫瑰、郁金香）無本地替代品。

**問題 1**：企業應該採用什麼庫存策略應對本地供應的不穩定性？

**答案要點**：
- 增加安全庫存（對易保存品種）
- 為缺乏替代品的品種建立戰略庫存
- 對於易腐爛品種，與花農簽訂更靈活的訂購協議
- 考慮建立本地乾花、永生花等保存時間長的替代品

**問題 2**：如何應用延遲策略來降低鮮花腐損？

**答案要點**：
- 採購鮮花後不立即配送到花店
- 存放在統一的冷鏈倉庫
- 根據花店的實時訂單，在最後48小時內進行分揀與配送
- 減少花店端的庫存時間，降低腐損率

---

### 情境二：汽車零部件供應商的多元化困境

背景：一家汽車製造商向供應商甲採購關鍵零件（變速器），佔全球產能的80%。為了應對風險，企業決定建立第二供應商乙。

**問題 1**：企業應該預留多少產能給第二供應商？

**答案要點**：
- 若完全替換：需要20-30%（供應商甲不願意失去主訂單，而且切換成本高）
- 實務做法：首先預留10-15%的訂單給乙測試
- 6個月後根據乙的績效逐步增加至20-25%
- 保持供應商甲的80%+以維持成本競爭力

**問題 2**：這個多元化戰略的成本代價是什麼？

**答案要點**：
- 製造成本增加：供應商乙的產能規模小，成本可能高10-20%
- 管理成本增加：需要維護兩套質檢、物流體系
- 溝通成本：協調兩家供應商的交期、品質標準
- 預計年增成本：2000-5000萬元（具體視訂購規模）

---

### 情境三：電商配送的速度與成本權衡

背景：一家電商平台在華東地區與3家物流公司合作，目標是實現「48小時配送」。

**當前情況**：
- 物流成本：12元/件
- 配送速度：48-72小時
- 客戶滿意度：80%

**優化目標**：達成「24小時配送」

**問題**：實現24小時配送需要什麼供應鏈改變？成本增加多少？

**答案要點**：

1. **倉儲佈局**：
   - 從3個區域倉庫 → 新增8-10個小型分倉
   - 成本：年增1000-2000萬元（倉租、人力、系統）

2. **庫存預置**：
   - 將熱銷商品（佔SKU的20%、銷量的80%）提前分散存放
   - 資金占用增加：5000-1億元

3. **運輸優化**：
   - 專線配置：市區內24小時內快配人員
   - 與第4家快速物流商合作（成本更高，服務更快）
   - 物流成本增加：5-10元/件 → 新成本17-22元/件

4. **資訊系統**：
   - 實時庫存可見性、智慧派單、路線優化
   - 系統投資：2000-3000萬元

**成本總結**：
- 年增運營成本：5000-7000萬元
- 在此基礎上，客戶滿意度預計提升至90-95%
- 決策點：這5000-7000萬的投資能帶來多少額外銷售與品牌溢價

## 計算題（3-4題）

### 計算題一：安全庫存與服務水準

**題目**：
某快消品公司銷售牙膏，年銷量120,000件，日銷量均勻，標準差50件/天。採購提前期為14天。

要求：
1. 計算不同服務水準下的安全庫存
2. 如果倉儲成本為1元/件/年，比較不同服務水準的庫存成本代差

**已知**：
- Z(90%) = 1.28
- Z(95%) = 1.65
- Z(99%) = 2.33

**解答**：

公式：安全庫存 = Z × σ × √L

其中 L = 14天，σ = 50件/天

1. **安全庫存計算**：
```
90%服務水準：
SS₉₀ = 1.28 × 50 × √14 = 1.28 × 50 × 3.74 = 239件

95%服務水準：
SS₉₅ = 1.65 × 50 × √14 = 1.65 × 50 × 3.74 = 308件

99%服務水準：
SS₉₉ = 2.33 × 50 × √14 = 2.33 × 50 × 3.74 = 435件
```

2. **庫存成本對比**：
```
90%：年持有成本 = 239 × 1元 = 239元
95%：年持有成本 = 308 × 1元 = 308元
99%：年持有成本 = 435 × 1元 = 435元

成本差異：
- 從90%→95%增加69元（9%的安全庫存提升）
- 從95%→99%增加127元（14%的安全庫存提升）
```

3. **決策建議**：
對於標準化、重複購買的日用品，95%的服務水準通常是最優選擇，平衡缺貨成本與庫存成本。

### 計算題二：經濟訂購量（EOQ）

**題目**：
製造商每年採購100,000件零件。每次訂購成本（處理、檢驗、物流）為500元。零件年度單位持有成本為20元/件。

要求：計算經濟訂購量與最優訂購策略

**解答**：

公式：EOQ = √(2×D×S / H)

代入：D = 100,000，S = 500，H = 20

```
EOQ = √(2 × 100,000 × 500 / 20)
    = √(100,000,000 / 20)
    = √5,000,000
    = 2,236件
```

**最優訂購策略**：
- 每次訂購量：2,236件
- 年訂購次數：100,000 ÷ 2,236 ≈ 45次
- 訂購間隔：365 ÷ 45 ≈ 8天

**成本計算**：
```
年度訂購成本 = (D/Q) × S = (100,000/2,236) × 500 = 22,359元
年度持有成本 = (Q/2) × H = (2,236/2) × 20 = 22,360元
總成本 = 22,359 + 22,360 = 44,719元
```

**對比非最優方案**：
```
若訂購量為1,000件/次：
- 訂購成本：(100,000/1,000) × 500 = 50,000元
- 持有成本：(1,000/2) × 20 = 10,000元
- 總成本：60,000元

若訂購量為5,000件/次：
- 訂購成本：(100,000/5,000) × 500 = 10,000元
- 持有成本：(5,000/2) × 20 = 50,000元
- 總成本：60,000元

結論：EOQ在2,236件時成本最低（44,719元），比其他方案節省約25%
```

### 計算題三：長鞭效應量化分析

**題目**：
零售商的月銷售數據如下：
```
月份    實際銷量
1月     1000件
2月     1050件
3月     950件
4月     1100件
5月     900件
```

但批發商根據訂單情況，估計增長率為：
```
1月預估：+5% → 訂貨量1050件
2月預估：+5% → 訂貨量1103件
3月預估：-10% → 訂貨量993件
4月預估：+15% → 訂貨量1142件
5月預估：-20% → 訂貨量914件
```

要求：計算長鞭效應比例

**解答**：

1. **計算需求波動**：
```
實際銷量波動 = (最高值 - 最低值) / 平均值 × 100%
平均銷量 = (1000+1050+950+1100+900) / 5 = 1000件
波動範圍 = 1100 - 900 = 200件
波動率 = 200 / 1000 = 20%
```

2. **計算訂單波動**：
```
訂貨量波動 = (最高值 - 最低值) / 平均值 × 100%
平均訂貨 = (1050+1103+993+1142+914) / 5 = 1040.4件
波動範圍 = 1142 - 914 = 228件
波動率 = 228 / 1040.4 = 21.9%
```

3. **長鞭效應比例**：
```
長鞭倍數 = 訂單波動率 / 需求波動率 = 21.9% / 20% ≈ 1.1倍

說明：
- 實際銷量波動20%
- 批發商訂單波動21.9%（略高於實際）
- 倍數接近1，說明長鞭效應不明顯

如果加入更多層級（製造商、供應商），波動會進一步放大：
- 製造商可能看到30%的波動
- 供應商可能看到50%以上的波動
```

### 思考題一：供應鏈管理在ESG時代的新挑戰

**題目**：
企業承諾在2030年實現「碳中和供應鏈」。但目前主要供應商位於成本低廉的發展中國家，這些地區：
- 能源仍以煤炭為主
- 環保法規寬鬆
- 轉向綠色能源成本高達30-50%

**問題**：
1. 企業應該如何在成本競爭與ESG承諾之間平衡？
2. 如果不願意承擔綠色轉型成本，企業會面臨什麼長期風險？

**延伸思考方向**：
- 與供應商共擔綠色投資成本（降價幅度換換環保投資）
- 尋求替代供應商（成本上升但環保達標）
- 轉向本地化採購（接近綠色能源發達的區域）
- 通過品牌溢價與消費者溝通，抵消成本上升
- 在供應鏈中尋找效率提升機會，用節省的成本抵銷綠色投資

**評估標準**：
- 考慮了多方利益相關者（供應商、消費者、環境）
- 提出了現實可行方案，而非純理想願景
- 量化成本與收益的權衡

### 思考題二：地緣政治風險下的供應鏈戰略

**題目**：
一家消費電子企業依賴台灣供應商供應80%的核心零件。隨著地緣政治風險上升，企業考慮多元化供應源，但替代供應商有以下特點：

- **日本供應商**：成本高20%，品質等同，但政治風險更低
- **韓國供應商**：成本高15%，品質略低，需要3-6個月認證期
- **越南供應商**：成本高10%，品質需觀察，規模小但發展潛力大
- **美國供應商**：成本高30%，品質優秀，獲得政策支持

**問題**：
企業應該採用什麼供應商多元化策略？考慮以下因素：
1. 長期成本
2. 風險承受能力
3. 技術發展前景
4. 政策與地緣政治變化

**延伸思考方向**：
- 產品線與供應源的對應（高利潤產品用最安全的供應源）
- 時間推進（逐步過渡 vs 快速轉移）
- 供應商扶持（幫助新供應商提高能力，長期節省成本）
- 技術替代（尋求不同技術路線，減少對特定供應商的依賴）

### 思考題三：最後一哩配送的優化困局

**題目**：
電商企業提供「24小時送達」服務，成本為15元/件。但客戶投訴率高：
- 15%的單件被誤送或遺失
- 配送人員服務態度差
- 投訴處理周期長

降低配送成本的方案A（眾包配送，成本8元/件）會進一步惡化服務品質。

**問題**：
1. 企業應該如何在成本與服務品質之間做出決策？
2. 有沒有「第三條路」——既降低成本又提升品質？

**延伸思考方向**：
- 差異化配送（不同客戶不同方案）
- 技術賦能（無人配送、智慧分揀）
- 客戶預期管理（改變消費者對速度的預期）
- 模式創新（客戶到驛站自取 vs 上門配送）
- 數據分析（找出高風險的配送環節與地點，針對優化）

**評估標準**：
- 真實理解消費者需求，而非盲目追求速度
- 提出可量化的成本-效益分析
- 考慮長期品牌價值，而非短期成本節省
